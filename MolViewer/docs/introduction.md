Molecule Viewer (MolViewer) is a **currently uncompleted** molecular visualization tool.

<!--more-->

## What is it

MolViewer is a tool which combines molecular visualization and the advantages of MATLAB processing matrices.

## What does it do

MolViewer can display molecule, visulaize chemistry-related files and convert files between common file types.
