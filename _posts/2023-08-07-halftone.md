在Halftone风格图片中，圆点的半径取决于此点的灰度值，深色位置圆点半径大二浅色位置圆点半径小。因此当采样足够密集时灰度接近连续图片。

# 实现

程序需要的库如下：

```python
from PIL import Image
import matplotlib.colors
import numpy as np
```

绘制svg的函数代码如下，`halftone`函数接受3个参数，第一个参数为原图片的路径；第二个参数为新生成图片的宽度（x方向圆点的数量，高度由原图片宽高比自动设置）；第三个参数为圆点的颜色，当`color`设置为HEX格式的颜色时，输出的图片中圆点颜色为`color`，当`color`被设置为`'rgb'`时，输出图片中圆点的颜色由原图片该位置对应颜色所决定。

```python
def halftone(path,width,color):
    img=openimg(path,width)
    if np.any(img>1):
        img=img/255
    
    grayimg=rgb2gray(img)
    radius=gray2radius(grayimg)
    if color=='rgb':
        R=img[:,:,0].flatten()
        G=img[:,:,1].flatten()
        B=img[:,:,2].flatten()
        color=np.vstack((R,G,B)).T
        color=rgb2hex(color)
    else:
        color=[color for i in range(len(radius))]
   
    x,y=np.meshgrid(np.arange(img.shape[1]),np.arange(img.shape[0]))
    x=x.flatten()
    y=y.flatten()
    head=['<svg viewBox="0 0 '+str(8*img.shape[1])+' '+str(8*img.shape[0])+'" xmlns="http://www.w3.org/2000/svg">\n']
    bottom=['</svg>']
    data_list=[]
    for i in range(len(radius)):
        data='  <circle cx="'+str(8*x[i]+4)+'" cy="'+str(8*y[i]+4)+'" r="'+str(round(radius[i],2))+'" fill="'+color[i]+'" />\n'      
        data_list.append(data)
    
    svg_data=head+data_list+bottom
    svg_file=open("Image.svg","w")
    svg_file.writelines(svg_data)
    svg_file.close()
```

该函数首先调用`openimg`函数（函数源代码见后文），`openimg`接受文件路径与图片宽度为参数，返回`Array of uint8`矩阵，矩阵的列数为指定宽度，行数依据输入图片的宽高比自动设置。随后将RGB矩阵归一化到 $[0\sim1]$ 的范围内。

第二步利用`rgb2gray`和`gray2radius`函数决定Halftone图片中圆点的半径，圆点的半径取决于此点的灰度值（源代码见后文）。并依据`color`的设置确定每个圆点的颜色。

第三步利用`meshgrid`函数生成Halftone图片中每个圆点的圆心位置。最后依据圆心位置、半径与颜色为svg添加circle并写入文件。

## 子函数

```python
def openimg(path,width):
    img=Image.open(path)
    height=int(img.size[1]*width/img.size[0])
    img=img.resize((width, height))
    img=np.array(img)
    return img
```

```python
def rgb2gray(rgb):
    gray=0.299*rgb[:,:,0]+0.587*rgb[:,:,1]+0.114*rgb[:,:,2]
    return gray
```

```python
def gray2radius(gray):
    data=gray.flatten()
    radius=4*np.power(1-data,0.3)
    return radius
```

```python
def rgb2hex(rgb):
    hex_list=[]
    for i in range(len(rgb)):
        hex_list.append(matplotlib.colors.to_hex(rgb[i,:]))
    return hex_list
```
